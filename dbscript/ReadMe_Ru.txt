                                  DBScripter                                  
                                  ~~~~~~~~~~

Эта утилита предназначена для скриптования данных из таблиц в текстовый файл. 
Созданный скрипт содержит набор операторов INSERT и прочих и полностью готов к
выполнению в контексте целевой базы данных.

- Поддерживает MS SQL 2000-2008
- Создает скрипт в ANSI или Unicode
- Может добавлять оператор DELETE в начало скрипта для очистки таблицы
- Может добавлять операторы ENABLE/DISABLE TRIGGER для отключения триггеров на
  время заполнения таблицы
- Автоматически добавляет SET IDENTITY_INSERT ON для таблиц, имеющих поле
  identity
- Опционально может добавлять выражение DROP CONSTRAINT (с пересозданием 
  ограничений в конце скрипта) для предотвращения конфликтов при чистке 
  связанных таблиц
- Вы можете задать подходящий формат для полей дата-время
- Также вы можете задать условие выборки для каждой таблицы (без WHERE в 
  начале)  

Системные требования
Поддерживаемые ОС: Windows 2000 SP3 или выше; Windows 98 или выше
.NET Framework 2.0
